# Student Dashboard (Next.js + Supabase)

A modern SaaS-style student learning dashboard built with:

- Next.js App Router
- Supabase (PostgreSQL)
- Tailwind CSS
- Framer Motion

## Features
- Bento grid dashboard UI
- Live course data from Supabase
- Animated progress tracking
- Responsive design
- Mobile bottom navigation

## Setup
1. Install dependencies
2. Add `.env.local`
3. Run `npm run dev`

## Deployment
Deployed on Vercel
